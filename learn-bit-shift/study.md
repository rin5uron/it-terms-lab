# сѓисЃЋсЃѕТ╝ћу«ЌсЂ«УДБУфгТДІТѕљ№╝ѕТюђухѓТДІТѕљТАѕ№╝Ѕ

---

## РЉа сѓисЃЋсЃѕТ╝ћу«ЌсЂесЂ»№╝Ъ

сѓисЃЋсЃѕТ╝ћу«ЌсЂесЂ»сђЂ**2жђ▓ТЋ░сЂ«сЃЊсЃЃсЃѕ№╝ѕТАЂ№╝ЅсѓњтидтЈ│сЂФсЂџсѓЅсЂЎсЂЊсЂесЂДсђЂ2сЂ«у┤»С╣ЌсЂФсѓѕсѓІТјЏсЂЉу«ЌсЃ╗тЅ▓сѓіу«ЌсѓњсѓисЃ│сЃЌсЃФсЂІсЂцжФўжђЪсЂФУАїсЂєТ╝ћу«Ќ**сЂДсЂЎсђѓ

---

## РЉА 10жђ▓ТЋ░сЂДУђЃсЂѕсЂдсЂ┐сѓІ

```
123 << 1 Рєњ 1230№╝ѕ├Ќ10№╝Ѕ
123 >> 1 Рєњ 12.3№╝ѕ├и10№╝Ѕ

Рєњ ТАЂсѓњсЂџсѓЅсЂЎсЂЊсЂесЂДсђЂ10тђЇсЃ╗10тѕєсЂ«1сЂФсЂфсѓІ
```

сЂЊсЂ«сђїТАЂсѓњсЂџсѓЅсЂЎ№╝ЮСйЋтђЇсЃ╗СйЋтѕєсЂ«1сђЇсЂесЂёсЂєТёЪУдџсѓњсђЂ  
**2жђ▓ТЋ░сЂ«СИќуЋїсЂДсѓѓтљїсЂўсѓѕсЂєсЂФт┐юућесЂЎсѓІ**сЂ«сЂїсђїсѓисЃЋсЃѕТ╝ћу«ЌсђЇсЂДсЂЎсђѓ

---

## РЉб 2жђ▓ТЋ░сЂДсЂ«сѓисЃЋсЃѕТ╝ћу«Ќ

сѓисЃЋсЃѕТ╝ћу«ЌсЂїсђї2сЂ«у┤»С╣ЌсЂФсѓѕсѓІТјЏсЂЉу«ЌсЃ╗тЅ▓сѓіу«ЌсђЇсЂесЂЌсЂдТЕЪУЃйсЂЎсѓІсЂ«сЂ»сђЂ**2жђ▓ТЋ░сЂасЂІсѓЅсЂЊсЂЮ**ТѕљсѓіуФІсЂАсЂЙсЂЎсђѓ
10жђ▓ТЋ░сЂДсЂ»ТАЂсѓњсЂџсѓЅсЂЎсЂе10тђЇсЃ╗10тѕєсЂ«1сЂФсЂфсѓІсѓѕсЂєсЂФсђЂ2жђ▓ТЋ░сЂДсЂ»ТАЂсѓњсЂџсѓЅсЂЎсЂе2тђЇсЃ╗2тѕєсЂ«1сЂФсЂфсѓісЂЙсЂЎсђѓ
сЂЊсѓїсЂ»сђЂтљёТАЂсЂ«жЄЇсЂ┐сЂї2сЂ«у┤»С╣ЌсЂДУАеуЈЙсЂЋсѓїсѓІ2жђ▓ТЋ░сЂ«уЅ╣ТђДсЂФсѓѕсѓІсѓѓсЂ«сЂДсЂЎсђѓ

---

## РЉБ УФќуљєсѓисЃЋсЃѕ№╝ѕLogical Shift№╝Ѕ

**Рєњ ТАЂсѓњтидтЈ│сЂФсЂџсѓЅсЂЌсђЂуЕ║сЂёсЂЪТАЂсЂФсђї0сђЇсѓњУЕ░сѓЂсѓІТќ╣Т│Ћсђѓ**

­ЪЊї СИ╗сЂФТГБсЂ«ТЋ┤ТЋ░сЃ╗жЮъУ▓аТЋ┤ТЋ░сЂДСй┐сѓЈсѓїсђЂ**тидсѓисЃЋсЃѕсЂД├Ќ2сђЂтЈ│сѓисЃЋсЃѕсЂД├и2** сЂФсЂфсѓісЂЙсЂЎсђѓ

### ­Ъћ╣ СЙІ№╝џУФќуљєсѓисЃЋсЃѕ№╝ѕ8сЃЊсЃЃсЃѕУАеуЈЙ№╝Ѕ

```
# тидсѓисЃЋсЃѕ (<<): уЕ║сЂёсЂЪТАЂсЂФ0сѓњУЕ░сѓЂсѓІ
# 5 (0000 0101) сѓњ1сЃЊсЃЃсЃѕтидсѓисЃЋсЃѕ
тЁЃсЂ«тђц:  0 0 0 0  0 1 0 1  (5)
тидсѓисЃЋсЃѕ: 0 0 0 0  1 0 1 0  (10)  <- тЈ│уФ»сЂФ0сЂїУ┐йтіасЂЋсѓїсѓІ

# 5 (0000 0101) сѓњ2сЃЊсЃЃсЃѕтидсѓисЃЋсЃѕ
тЁЃсЂ«тђц:  0 0 0 0  0 1 0 1  (5)
тидсѓисЃЋсЃѕ: 0 0 0 1  0 1 0 0  (20)  <- тЈ│уФ»сЂФ0сЂї2сЂцУ┐йтіасЂЋсѓїсѓІ

# тЈ│сѓисЃЋсЃѕ (>>): уЕ║сЂёсЂЪТАЂсЂФ0сѓњУЕ░сѓЂсѓІ
# 8 (0000 1000) сѓњ1сЃЊсЃЃсЃѕтЈ│сѓисЃЋсЃѕ
тЁЃсЂ«тђц:  0 0 0 0  1 0 0 0  (8)
тЈ│сѓисЃЋсЃѕ: 0 0 0 0  0 1 0 0  (4)   <- тидуФ»сЂФ0сЂїУ┐йтіасЂЋсѓїсѓІ

# 8 (0000 1000) сѓњ2сЃЊсЃЃсЃѕтЈ│сѓисЃЋсЃѕ
тЁЃсЂ«тђц:  0 0 0 0  1 0 0 0  (8)
тЈ│сѓисЃЋсЃѕ: 0 0 0 0  0 0 1 0  (2)   <- тидуФ»сЂФ0сЂї2сЂцУ┐йтіасЂЋсѓїсѓІ
```

Рђ╗ УФќуљєтЈ│сѓисЃЋсЃѕсЂДсЂ»сђЂтИИсЂФтидуФ»сЂФ **0** сЂїтЁЦсѓісЂЙсЂЎсђѓ
Рђ╗ У▓асЂ«ТЋ░сЂФсЂ»т»Йт┐юсЂЌсЂдсЂёсЂЙсЂЏсѓЊсђѓ

---

## РЉц у«ЌУАЊсѓисЃЋсЃѕ№╝ѕArithmetic Shift№╝Ѕ

**Рєњ тидуФ»сЂ«угдтЈисЃЊсЃЃсЃѕсѓњтЏ║т«џсЂЌсЂЪсЂЙсЂЙтЈ│сѓисЃЋсЃѕсЂЎсѓІТќ╣Т│Ћсђѓ**

- У▓асЂ«ТЋ░сѓѓсЂЮсЂ«сЂЙсЂЙТЅ▒сЂѕсѓІсЂЪсѓЂсђЂугдтЈиС╗ўсЂЇТЋ┤ТЋ░сЂ«жЎцу«ЌсЂФсѓѕсЂЈућесЂёсѓЅсѓїсЂЙсЂЎсђѓ
- CPUсЃгсЃЎсЃФсЂДсЂ»сђїугдтЈиС╗ўсЂЇТЋ┤ТЋ░№╝ѕsigned int№╝ЅсђЇсЂ«Т╝ћу«ЌсЂФСй┐сѓЈсѓїсЂЙсЂЎсђѓ

### ­Ъћ╣ СЙІ№╝џу«ЌУАЊтЈ│сѓисЃЋсЃѕ№╝ѕугдтЈиС╗ўсЂЇ8сЃЊсЃЃсЃѕУАеуЈЙ№╝Ѕ

```
# У▓асЂ«ТЋ░ -4 (1111 1100) сѓњ1сЃЊсЃЃсЃѕу«ЌУАЊтЈ│сѓисЃЋсЃѕ
# угдтЈисЃЊсЃЃсЃѕ (СИђуЋфтидсЂ«сЃЊсЃЃсЃѕ) сЂ»уХГТїЂсЂЋсѓїсѓІ
тЁЃсЂ«тђц:  1 1 1 1  1 1 0 0  (-4)
тЈ│сѓисЃЋсЃѕ: 1 1 1 1  1 1 1 0  (-2)  <- угдтЈисЃЊсЃЃсЃѕсЂ«1сЂїтидуФ»сЂФУцЄУБйсЂЋсѓїсѓІ
```

---

## РЉЦ УБюУХ│№╝џсѓфсЃ╝сЃљсЃ╝сЃЋсЃГсЃ╝№╝ѕтидсѓисЃЋсЃѕсЂДсѓѓУ▓асЂФсЂфсѓІсЂЊсЂесЂїсЂѓсѓІ№╝Ѕ

> тидсѓисЃЋсЃѕсЂДсѓѓсђЂ**ТюђСИіСйЇсЃЊсЃЃсЃѕсЂФ1сЂїтЁЦсѓІсЂеугдтЈисЂїтцЅсѓЈсѓІ**

### СЙІ№╝џ

```
# 64 (0100 0000) сѓњ1сЃЊсЃЃсЃѕтидсѓисЃЋсЃѕ
тЁЃсЂ«тђц:  0 1 0 0  0 0 0 0  (64)
тидсѓисЃЋсЃѕ: 1 0 0 0  0 0 0 0  (-128) <- ТюђСИіСйЇсЃЊсЃЃсЃѕсЂї1сЂФсЂфсѓісђЂугдтЈисЂїтЈЇУ╗б
```

Рєњ тидсѓисЃЋсЃѕсЂДсѓѓУ▓асЂФсЂфсѓІ№╝ЂсЂЊсѓїсЂ» **ТАЂсЂѓсЂхсѓї№╝ѕсѓфсЃ╝сЃљсЃ╝сЃЋсЃГсЃ╝№╝Ѕ** сЂФсѓѕсѓІсѓѓсЂ«сЂДсЂЎсђѓ

---

## РЉд Т│еТёЈТЏИсЂЇ№╝ѕжђ▓ТЋ░сЂесѓисЃЋсЃѕТ╝ћу«Ќ№╝Ѕ

сѓисЃЋсЃѕТ╝ћу«ЌсЂ»сђЂ**2жђ▓ТЋ░сЂ«сЃЊсЃЃсЃѕсѓњуЏ┤ТјЦТЊЇСйюсЂЎсѓІТ╝ћу«Ќ**сЂДсђЂ  
СИ╗сЂФ **CPUтєЁжЃесѓёСйјсЃгсѓцсЃцсЃ╝тЄдуљє**сЂДСй┐сѓЈсѓїсЂЙсЂЎсђѓ

8жђ▓ТЋ░сѓё16жђ▓ТЋ░сЂДсЂ» **УАеУеўсѓњу░АуЋЦтїќсЂЎсѓІуЏ«уџёсЂДСй┐сѓЈсѓїсѓІсЂЊсЂесЂ»сЂѓсѓісЂЙсЂЎ**сЂїсђЂ  
**Т╝ћу«ЌУЄфСйЊсЂ»тИИсЂФ2жђ▓ТЋ░сЂФТѕ╗сЂЌсЂдтЄдуљєсЂЋсѓїсЂЙсЂЎсђѓ**
