基数変換 練習問題

実際に問題を解いて理解を深めよう

レベル1：基本問題

問1：10進数 → 2進数

10進数の25を2進数に変換してください。

解答

解法：2で割って余りを逆順に並べる

25 ÷ 2 = 12 ... 1
12 ÷ 2 = 6 ... 0
6 ÷ 2 = 3 ... 0
3 ÷ 2 = 1 ... 1
1 ÷ 2 = 0 ... 1

→ 逆順に読む → 11001（答え）

問2：2進数 → 10進数

2進数の1101を10進数に変換してください。

解答

解法：各桁に2の累乗をかけて合計する

1×2³ + 1×2² + 0×2¹ + 1×2⁰
= 1×8 + 1×4 + 0×2 + 1×1
= 8 + 4 + 0 + 1 = 13（答え）

問3：10進数 → 16進数

10進数の255を16進数に変換してください。

解答

解法：16で割って余りを逆順に並べる（10以上はA〜Fに変換）

255 ÷ 16 = 15 ... 15（F）
15 ÷ 16 = 0 ... 15（F）

→ 逆順に読む → FF（答え）

レベル2：応用問題

問4：16進数 → 2進数

16進数のA3を2進数に変換してください。

解答

解法：各桁を4ビットの2進数に変換

A = 10 → 1010（2進数）
3 = 3 → 0011（2進数）

→ つなげる → 10100011（答え）

問5：小数の変換

10進数の0.625を2進数に変換してください。

解答

解法：2を掛けて整数部分を左から順に読む

0.625 × 2 = 1.25 → 1
0.25 × 2 = 0.5 → 0
0.5 × 2 = 1.0 → 1

→ 左から読む → 0.101（答え）

問6：2進数から色コードを求める

2進数のRGB値「11111111 00000000 11111111」を色コードで表したものはどれか。

ア：#FF00FF　イ：#FF0000　ウ：#00FFFF　エ：#FFFF00

解答

解法：2進数を16進数に変換して色コードを作る

11111111 = 255 → FF（16進数）
00000000 = 0 → 00（16進数）
11111111 = 255 → FF（16進数）

→ RGB(255, 0, 255) = #FF00FF（マゼンタ色）

答え：ア

レベル3：実践問題

問7：メモリアドレス

メモリアドレス 0x1A2B を2進数に変換してください。

解答

解法：各桁を4ビットの2進数に変換

1 = 1 → 0001
A = 10 → 1010
2 = 2 → 0010
B = 11 → 1011

→ つなげる → 0001101000101011（答え）

問8：複合変換

2進数の10110110を16進数に変換してください。

解答

解法：4桁ずつ区切って16進数に変換

1011 = 11 → B
0110 = 6 → 6

→ つなげる → B6（答え）

ア：0.101　イ：0.110　ウ：0.111　エ：1.101

解答

0.25 × 2 = 0.5 → 0
0.5 × 2 = 1.0 → 1

→ 左から読む → 0.101

問10：平成29年秋期

16進数の小数0.248を10進数の分数で表したものはどれか。

ア：31/32　イ：73/256　ウ：9/32　エ：41/128

解答

解法：各桁に16の累乗をかけて合計する

0.248 = 2×16⁻¹ + 4×16⁻² + 8×16⁻³
= 2×(1/16) + 4×(1/256) + 8×(1/4096)
= 2/16 + 4/256 + 8/4096
= 512/4096 + 64/4096 + 8/4096
= 584/4096 = 73/512

答え：イ（73/256）

問11：平成28年秋期

2進数の101.011を10進数で表したものはどれか。

ア：5.375　イ：5.625　ウ：5.75　エ：5.875

解答

解法：各桁に2の累乗をかけて合計する

101.011 = 1×2² + 0×2¹ + 1×2⁰ + 0×2⁻¹ + 1×2⁻² + 1×2⁻³
= 1×4 + 0×2 + 1×1 + 0×0.5 + 1×0.25 + 1×0.125
= 4 + 0 + 1 + 0 + 0.25 + 0.125
= 5.375

答え：ア

問12：平成27年秋期

10進数の-1.625を2進数で表したものはどれか。

ア：-1.101　イ：-1.110　ウ：-10.101　エ：-10.110

解答

解法：まず1.625を2進数に変換してから負の符号を付ける

1.625の整数部：1 → 1
1.625の小数部：
0.625 × 2 = 1.25 → 1
0.25 × 2 = 0.5 → 0
0.5 × 2 = 1.0 → 1

1.625 = 1.101
-1.625 = -1.101

答え：ア

問13：平成26年秋期

10進数の0.375を2進数で表したものはどれか。

ア：0.011　イ：0.101　ウ：0.110　エ：0.111

解答

解法：2を掛けて整数部分を左から順に読む

0.375 × 2 = 0.75 → 0
0.75 × 2 = 1.5 → 1
0.5 × 2 = 1.0 → 1

→ 左から読む → 0.011

答え：ア

問14：平成25年秋期

16進数の小数0.8を10進数の分数で表したものはどれか。

ア：1/2　イ：1/3　ウ：1/4　エ：1/5

解答

解法：各桁に16の累乗をかけて合計する

0.8 = 8×16⁻¹
= 8×(1/16)
= 8/16 = 1/2

答え：ア

問15：平成24年秋期

10進数の13を8進数で表したものはどれか。

ア：15　イ：16　ウ：17　エ：18

解答

解法：8で割って余りを逆順に並べる

13 ÷ 8 = 1 ... 5
1 ÷ 8 = 0 ... 1

→ 逆順に読む → 15

答え：ア

間違いやすいポイント

  • 桁の重み：一の位は0乗から始まる（2⁰ = 1）
  • 16進数：A=10, B=11, C=12, D=13, E=14, F=15
  • 小数変換：無限に続く場合がある（例：0.1）
  • 色コード：RGBの順序は赤→緑→青
