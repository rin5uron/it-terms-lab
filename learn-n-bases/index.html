なぜ？
電気信号の「オン / オフ」で動いているから

  • オン（電流あり）→ 1
  • オフ（電流なし）→ 0

つまり
2進数はコンピューターにとってシンプルかつ確実な計算方法

③ n進数とは？

「n個の数字（0〜n−1）を使って数を表す方式」のことです。

ポイント①：使える数字はn個
10進数０−９の１０個
2進数０−１の２個
16進数０−９，A−Fの１６個
ポイント②基数nで桁上がり

  • 2進数：0,1（2で桁上がり）
  • 10進数：0〜9（10で桁上がり）
  • 16進数：0〜9, A〜F（16で桁上がり）

④ 桁の重みとは

桁の重みとは、各桁がどれだけの価値を持つかを表します。

例：10進数 1234 の場合

1234 = 1×10³ + 2×10² + 3×10¹ + 4×10⁰ = 1×1000 + 2×100 + 3×10 + 4×1 = 1000 + 200 + 30 + 4

各桁の重みの概念

  • 千の位（1）： 重みは10³ = 1000
  • 百の位（2）： 重みは10² = 100
  • 十の位（3）： 重みは10¹ = 10
  • 一の位（4）： 重みは10⁰ = 1

なぜ指数が0から始まるの？

  • 一の位は「何倍にもしていない」= 1倍 = 基数⁰
  • 十の位は「基数を1回かけた」= 基数¹
  • 百の位は「基数を2回かけた」= 基数²
覚え方： 一の位から基数の0乗、1乗、2乗...と増えていく！

⑤ 基数変換が試験で出る理由

コンピュータは日々、主に10進数と16進数を２進数に変換して計算しているから

🔹 なぜ 10進・2進・16進 を相互に変換が必要なのか？

10進数

人間が普段使う表記

2進数

コンピューターが内部で使う表記（オン/オフ）

16進数

2進数（4ビット）= 16通り → 相性抜群！
アドレス・色コードなどでよく使われる

色コード例：#FF0000（赤）
メモリアドレス例：0x1A2B3C4D

例： A3 → 1010 0011

実際の変換例

色コード #FF0000 を2進数に変換

#FF0000 = FF + 00 + 00 FF = 11111111 (2進数) 00 = 00000000 (2進数) 00 = 00000000 (2進数) 結果：11111111 00000000 00000000 （赤255、緑0、青0 = 赤色）

メモリアドレス 0x1A2B を2進数に変換

0x1A2B = 1 + A + 2 + B 1 = 0001 (2進数) A = 1010 (2進数) 2 = 0010 (2進数) B = 1011 (2進数) 結果：0001 1010 0010 1011

16進数と2進数の相性が良い理由

16進数の1桁 = 2進数の4桁に対応するため、変換が簡単です。

⑥ 実用のイメージ

コンピューターの数値処理の流れ

2進数（内部処理）
↓ 4ビット単位で変換
16進数（エンジニアが見るアドレスやデータ）
↓ 人に見せるときや計算
10進数（普段使う数字）

例えば、プログラムでメモリの場所を確認するときは16進数で表示されることが多く、それを人間が理解しやすいように10進数に変換したり、コンピューターの内部動作を理解するために2進数に変換したりします。

基数変換のやり方（試験頻出！）

① 10進数 → n進数

（1）整数部の変換

💡 nで割って、余りを下から順に並べる！ 基数Nで割ることで、桁ごとの文字列を求めている

例：13 を 2進数に変換する場合

13 ÷ 2 = 6 ... 1
6 ÷ 2 = 3 ... 0
3 ÷ 2 = 1 ... 1
1 ÷ 2 = 0 ... 1

→ 逆順に読む → 1101（答え）

（2）小数部の変換

💡 nを掛けて、出てきた整数部分を左から順に読む！ 基数Nで割ることで、桁ごとの文字列を求めている

例：0.625 を 2進数に変換する場合

0.625 × 2 = 1.25 → 1
0.25 × 2 = 0.5 → 0
0.5 × 2 = 1.0 → 1

→ 左から読む → 0.101（答え）

② n進数 → 10進数

💡 各桁に桁の重みをかけて、それぞれを合計する！

例1：2進数 1101 を 10進数に変換

1×2³ + 1×2² + 0×2¹ + 1×2⁰
= 8 + 4 + 0 + 1 = 13

例2：16進数 1A を 10進数に変換

1×16¹ + A×16⁰ = 1×16 + 10×1 = 26
（※A=10、B=11、... F=15）

✔ 試験でよく出る変換パターン

出題形式 方法（ざっくり）
10進 → 2進割って余りを逆に並べる
2進 → 10進桁ごとに 2ⁿ をかけて足す
10進 → 16進割って余り（16）＋ A〜Fに変換
16進 → 2進各桁を4ビットの2進数に置き換え
2進 → 16進4桁ずつ区切って16進数に変換

💡補足：指数が0から始まる理由

  • 一の位は「何倍にもしていない」= 1倍 = 基数⁰
  • 指数の性質：
    aⁿ ÷ aⁿ = a⁰ = 1
    → だから n⁰ = 1

📝まとめ

  • 整数部は「割って余り」
  • 小数部は「掛けて整数を抜き出す」
  • n進数→10進数は「重み（nの累乗）をかけて足す」
  • 桁の意味を理解すれば応用も効く！
重要：
  • コンピューターが2進数を使うのは、電気的に最もシンプルだから
  • 試験では10・2・16進数の変換が基本
  • 16進数は2進数との相性が良く、現場でもよく使われている
