2. ブール代数の基本法則

論理式を簡略化するために使われる、ブール代数の重要な法則です。

ド・モルガンの法則

論理式の変形において非常に重要です。

  • ¬(A・B) = ¬A + ¬B
    「AかつB」の否定は、「Aの否定 または Bの否定」と等しい。
  • ¬(A + B) = ¬A・¬B
    「AまたはB」の否定は、「Aの否定 かつ Bの否定」と等しい。

その他の重要な法則

  • 交換法則: A・B = B・A | A+B = B+A
  • 結合法則: (A・B)・C = A・(B・C) | (A+B)+C = A+(B+C)
  • 分配法則: A・(B+C) = A・B + A・C | A+(B・C) = (A+B)・(A+C)
  • 吸収法則: A・(A+B) = A | A+(A・B) = A

4. 過去問演習

問題1 (基本情報技術者試験より)

論理式 A・B + A・C を簡略化したものはどれか。

ア: A・(B+C) イ: A+(B・C) ウ: B・(A+C) エ: C・(A+B)

答えと解説

答え: ア
これはブール代数の**分配法則** A・(B+C) = A・B + A・C を利用します。共通の因子であるAでくくることで、式を簡略化できます。

問題2 (基本情報技術者試験より)

入力Aと入力Bがともに1のときだけ、出力が0になる論理ゲートはどれか。

ア: AND イ: NOR ウ: NAND エ: XOR

答えと解説

答え: ウ
NANDゲートは、ANDゲートの出力を否定したものです。ANDゲートは入力が両方1のときに出力が1になるため、NANDゲートはその逆で、入力が両方1のときに出力が0になります。

問題3 (基本情報技術者試験より)

半加算器において、入力A=1, B=0 のときの出力として正しいものはどれか。

ア: 和=0, 桁上がり=0 イ: 和=0, 桁上がり=1 ウ: 和=1, 桁上がり=0 エ: 和=1, 桁上がり=1

答えと解説

答え: ウ
半加算器の和はXOR(A⊕B)、桁上がりはAND(A・B)で計算されます。
和: 1 ⊕ 0 = 1
桁上がり: 1・0 = 0
